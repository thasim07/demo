How to Download container?

>>docker pull busybox(this is a container)
tag: tagname of the container
 
------------
commands:

>>>docker image pull selenium/standalone-chrome   //used to pull the container
>>>docker container create selenium/standalone-chrome //used to create the container
//after that we will get a ID
>>> docker container start ID //start the container using ID
>>>docker container ps  //checking the status of the container only on running state
>>>docker container stop ID //used to stop the container
>>>docker container ps -a // -a shows all the container
>>>docker container rm ID  // to remove the container
